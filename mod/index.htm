调用api流程

  1. 获得用户统一身份认证信息(用户名+统一身份认证密码)

    username=213150000
        password=******

  2. 将认证信息和 APPID 发送至认证中心, 获取用户UUID

    POST: http://www.heraldstudio.com/uc/auth 
        DATA: user=213150000&password=******&appid=******
        RETURN: 010cd8bd637755999a5da376054753ac6d083f74

  3. 根据API要求, 发送相应参数和 UUID 至 API 接口

    POST: http://www.heraldstudio.com/api/nic  
        DATA:
            uuid=010cd8bd637755999a5da376054753ac6d083f74
        RETURN: 
        {
        “a”: { 
              “state”: “未开通”,
               “used”: “0 B” 
             }, 
         “web”: { 
                 “state”: “正常,在线”, 
                 “used”: “14.95 MB” 
                }, 
         “b”: { 
               “state”: “离线”, “
               used”: “371,856,152 B” 
             },
         “left”: “9.88 元” 
        }
        POST: http://www.heraldstudio.com/api/simsimi 
        DATA: uuid=010cd8bd637755999a5da376054753ac6d083f74&msg=你好
        RETURN: 亲爱的，我非常好，有你更美好

认证中心接口

授权认证(auth)
描述 获取用户唯一标识符 UUID
URL http://www.heraldstudio.com/uc/auth
Method POST
参数 含义
user 一卡通
password 统一身份认证密码
appid APPID
返回值 含义
用户唯一标识符（UUID） 请求成功
HTTPError 400 参数错误或appid不存在
HTTPError 401 授权失败(用户名密码不匹配)
UUID合法性检查
描述 检测当前 UUID 是否合法
URL http://www.heraldstudio.com/uc/check
Method POST
参数 含义
uuid 用户唯一标识符
appid 应用唯一标识符
返回值 含义
用户一卡通号 UUID合法
HTTPError 400 参数错误或
HTTPError 401 UUID不合法
更新用户信息(update)
描述 更新用户基本信息
URL http://www.heraldstudio.com/uc/update
Method POST
参数 含义
cardnum 一卡通
password 统一身份认证密码
*number 可选:学号
*pe_password 可选:跑操查询密码
*lib_username 可选:图书馆用户名
*lib_password 可选:图书馆密码
*card_query_password 可选:一卡通查询密码
返回值 含义
‘OK’ 更新成功
HTTPError 400 参数错误
HTTPError 401 用户密码错误
解除授权(deauth)
描述 注销 UUID, 注销后的 UUID 不能再被使用
URL http://www.heraldstudio.com/uc/deauth
Method POST
参数 含义
uuid 用户唯一标识符
返回值 含义
‘OK’ 注销成功
HTTPError 400 参数错误

统一API接口

调用API方式
描述 调用统一API接口
URL http://www.heraldstudio.com/uc/[API_NAME]
Method POST
参数 含义
uuid 用户唯一标识符
*[opt] API其它参数
返回值 含义
HTTP 200 API返回内容
HTTPError 400 参数错误
HTTPError 401 UUID不合法
HTTPError 404 APPID 请求次数耗尽
HTTPError 408 API 请求超时

场馆预约(yuyue)
描述 场馆预约
API_NAME lecture
method 操作名称
*para 操作对应的参数

method对应需要的参数

{
        'getDate':{
            'param':[]
        },
        'myOrder':{
            'param':[]
        },
        'cancelUrl':{
            'param':['id']
        },
        'getOrder':{
            'param':['itemId','dayInfo']
        },
        'judgeOrder':{
            'param':['itemId','dayInfo','time']
        },
        'getPhone':{
            'param':[]
        },
        'getFriendList':{
            'param':['cardNo']
        },
        'new':{
            'param':['orderVO.useMode','orderVO.useTime','orderVO.itemId','orderVO.phone','useUserIds','orderVO.remark']
        }
}
Example
POST: http://www.heraldstudio.com/api/yuyue
DATA: uuid=****&method=judgeOrder&&itemId=1&dayInfo=2016-07-01&time=14:00-15:00
RETURN: 010cd8bd637755999a5da376054753ac6d083f74
Tip:
useUserIds格式为["1234","5678"]

    },
