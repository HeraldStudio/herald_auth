调用API流程

  1. 获得用户统一身份认证信息(用户名+密码)

    user=213120498
    password=????????

  2. 将认证信息和 APPID 发送至认证中心, 获取用户UUID

    POST: http://herald.seu.edu.cn/uc/auth
    DATA: user=213120498&password=????????&appid=??????????

    RETURN: 010cd8bd637755999a5da376054753ac6d083f74

  3. 根据API要求, 发送相应参数和 UUID 至 API 接口

    POST: http://herald.seu.edu.cn/api/nic
    DATA: uuid=010cd8bd637755999a5da376054753ac6d083f74

    RETURN:
    { “a”: { “state”: “未开通”, “used”: “0 B” }, “web”: { “state”: “正常,在线”, “used”: “14.95 MB” }, “b”: { “state”: “离线”, “used”: “371,856,152 B” }, “left”: “9.88 元” }

    POST: http://herald.seu.edu.cn/api/simsimi
    DATA: uuid=010cd8bd637755999a5da376054753ac6d083f74&msg=你好

    RETURN: 亲爱的，我非常好，有你更美好

新增用户,更改用户基本信息

  1. 完善用户基础信息(学号,跑操查询密码,图书馆帐号和密码等)

    POST: http://herald.seu.edu.cn/uc/update
    DATA: 参见 函数说明

    RETURN: OK

  2. 注册新用户

    update auth 中任一方法被调用, 都会自动注册新用户

认证中心接口

授权认证(auth)

描述 获取用户唯一标识符 UUID
URI http://herald.seu.edu.cn/uc/auth
Method POST
参数 含义
user 一卡通
password 统一认证密码
appid APPID
返回值 含义
用户唯一标识符（UUID） 请求成功
HTTPError 400 参数错误
HTTPError 401 授权失败

注:UUID 暂不设过期时间,未经注销将一直有效

UUID合法性检查(check)

描述 检测当前 UUID 是否合法
URI http://herald.seu.edu.cn/uc/check
Method POST
参数 含义
uuid 用户唯一标识符
appid 应用唯一标识符
返回值 含义
用户一卡通号 UUID合法
HTTPError 400 参数错误
HTTPError 401 UUID不合法

更新用户信息(update)

描述 注销 UUID, 注销后的 UUID 不能再被使用
URI http://herald.seu.edu.cn/uc/update
Method POST
参数 含义
cardnum 一卡通
password 统一认证密码
*number 可选:学号
*pe_password 可选:跑操查询密码
*lib_username 可选:图书馆用户名
*lib_password 可选:图书馆密码
*card_query_password 可选:一卡通查询密码
返回值 含义
‘OK’ 更新成功
HTTPError 400 参数错误
HTTPError 401 用户密码错误

解除授权(deauth)

描述 注销 UUID, 注销后的 UUID 不能再被使用
URI http://herald.seu.edu.cn/uc/deauth
Method POST
参数 含义
uuid 用户唯一标识符
返回值 含义
‘OK’ UUID合法
HTTPError 400 参数错误

调用API(api)

描述 调用统一API接口
URI http://herald.seu.edu.cn/api/[API NAME]
Method POST
参数 含义
uuid 用户唯一标识符
*[opt] API其它参数
返回值 含义
HTTPError 404 APPID 请求次数耗尽
HTTPError 408 API 请求超时
HTTP 200 API返回内容
HTTPError 400 参数错误
HTTPError 401 UUID不合法

空闲教室指定查询(emptyroom)

描述及参数 空闲教室指定查询
API NAME emptyroom
arg1 校区(spl – 四牌楼; djq – 丁家桥; jlh – 九龙湖; all – 所有)
arg2 第几周(范围1-20 – 仅针对丁家桥校区第1学期和第2学期合并的情况，其他校区依旧为1-16)
arg3 星期几(范围1-7 – 对应星期一到星期日)
arg4 开始(范围1-13 – 指打算开始自习时为第几节课)
arg5 结束(范围1-13 – 指打算结束自习时为第几节课)

Return Json

["教一-109", "教一-311", ..., "教八-403"]

空闲教室快捷查询(emptyroom)

描述及参数 空闲教室指定查询
API NAME emptyroom
arg1 校区(spl – 四牌楼; djq – 丁家桥; jlh – 九龙湖; all – 所有)
arg2 时间(today – 今天; tomorrow – 明天)
arg3 开始(范围1-13 – 指打算开始自习时为第几节课)
arg4 结束(范围1-13 – 指打算结束自习时为第几节课)
